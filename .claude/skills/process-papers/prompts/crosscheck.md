# Crosscheck Instructions

Verify the claims made in each paper through three types of checking.

## Type 1: Verify Against Cited Sources

For each paper, identify the **top 3-5 most important claims** — claims that are central to the paper's conclusions or that would significantly change the paper's implications if false.

For each claim:
1. Identify which reference the paper cites to support it
2. Use WebSearch to find the cited source (search for the cited paper's title + authors)
3. Use WebFetch to read the cited source's abstract or full text
4. Assess whether the cited source actually supports the claim as stated

Format each as:
- **Claim:** [the claim]
- **Cited source:** [reference]
- **Verification:** [Supported | Partially supported | Not supported | Could not verify]
- **Note:** [brief explanation]

## Type 2: Cross-Paper Comparison (skip if only one paper)

Compare findings across all papers in the batch:
- **Agreements:** Claims or findings that multiple papers support
- **Contradictions:** Claims where papers disagree or present conflicting evidence
- **Complementary findings:** Where one paper's findings extend or contextualize another's

If only one paper is being analyzed, skip this section entirely.

## Type 3: Fact-Check via Web Search

For each paper, identify 1-2 key factual claims (statistics, benchmarks, dates, widely-cited facts) and verify them:
1. Use WebSearch to find independent sources
2. Check if the claim is accurate as stated

Format:
- **Claim:** [the factual claim]
- **Web verification:** [Confirmed | Disputed | Unverifiable]
- **Source:** [URL or description of confirming/disputing source]

## Output

Produce a crosscheck section per paper with all three types of verification results.
